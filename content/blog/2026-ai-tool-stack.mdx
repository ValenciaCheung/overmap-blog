每月跑过几百个 AI 工具,真正能留在 dock / 浏览器 tab 里的不到 10 个。这篇写下截至 2026 年 5 月,我每天会打开的那些 —— 也是这个站 [/tools](/tools) 里被打上 **编辑精选** 的那批。

## 写代码 · 三件套

### Cursor + Claude Code + v0

- **Cursor** —— 主力 IDE,Tab 补全、Composer 多文件改、Agent 跑长任务,工程师写 React/Go 全栈足够
- **Claude Code** —— CLI Agent,跟终端、git、shell 紧密集成,适合"跑一段命令、看结果、再决策"的工作流。CI 调试和本地小项目我都让它来
- **v0** —— 从 prompt 到可粘贴的 React + shadcn 组件,把"我想要个登录页"变成 5 分钟出原型

**为什么不是 Copilot?** Copilot 在企业里还是好用,但 Agent 模式被 Cursor / Claude Code 拉开太多。

## 写内容 · 一长一短

### Claude (长文) + Suno (音视频配乐)

- **Claude** —— 写正式邮件 / 写文档 / 改稿 / 总结长会议录音。长文本理解和语气把控比 ChatGPT 稳
- **Suno** —— 视频配乐找不到合适的、又不想给版权,Suno 出一段定制曲。V5 之后人声 demo 也直接能用

## 设计 · 三方拼图

### Midjourney + Krea + Magnific

- **Midjourney** —— 概念图、艺术海报、风格探索。V7 + Style Reference 让品牌一致性变现实
- **Krea** —— 实时画图(画线即生成),想做 hero banner 或 OG image 时直接刷,不用写 prompt
- **Magnific** —— 任何 AI 图最后都过一遍它,4x 放大 + 细节增强,印刷品可用

## 视频 · 两套并存

### Runway + Kling

- **Runway** —— 商业短视频,Act-One 让任意图片说话,可控性强
- **Kling** —— 镜头运动表现力第一档,2.5 之后开始替代 Sora 在我工作流的位置(Sora 还是贵)

## 知识管理 · 一个就够

### Granola

桌面 AI 会议笔记。打开 Granola → 开 Zoom → 自动转写 + 结构化 + 跟 Notion 同步。一个 app 替代了"会议记录 + 转写 + 总结提示词"三步。

## 通用问答 · 两个互补

### Perplexity + Claude

- **Perplexity** —— "查一件事"专用。实时联网 + 给引用,论文 / 行情 / 软件文档查询不打开 Google
- **Claude** —— 不查事实,只想要意见 / 改方案 / 解释概念的时候

## 我没在用的(但很多人推)

- **ChatGPT** —— 还在订阅,主要为了 Sora 和 Custom GPTs。日常对话用 Claude 更顺手
- **Notion AI** —— 已经退订,Granola + Claude 完全覆盖
- **Jasper / Copy.ai** —— 营销文案用 Claude 直接写,效果一样好,免一笔 SaaS 费

## 工具会越来越多,工作流不会

新工具每月几十个,我会试,但能留下来的标准没变 ——

1. **能不能替代我已有工作流里的某个环节?** 不能就不留
2. **价格 vs 节省时间** 算得过来吗?
3. **会不会被锁定?** Vendor lock-in 严重的(Notion AI 这种)优先级最低
